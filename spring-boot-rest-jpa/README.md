# Spring Boot REST API with JPA

A simple REST API for User Management built with Spring Boot, JPA, and MySQL.

This is a basic Spring Boot application that provides REST endpoints to manage users (Create, Read, Update, Delete). It uses MySQL database for data persistence and follows a clean layered architecture.

**Tech Stack**: Spring Boot 2.7.18 | Spring Data JPA | MySQL 8 | Lombok | Maven

## ğŸš€ Quick Start

### Prerequisites
- Java 8 or higher
- Maven 3.6+
- MySQL 8.0+

### 1. Setup Database
Create a MySQL database and user:
```sql
CREATE DATABASE testdb;
CREATE USER 'summarydb'@'localhost' IDENTIFIED BY 'Docker#Root99';
GRANT ALL PRIVILEGES ON testdb.* TO 'summarydb'@'localhost';
FLUSH PRIVILEGES;
```

### 2. Configure Database Connection
Edit `src/main/resources/application.properties` if needed:
```properties
spring.datasource.url=jdbc:mysql://127.0.0.1:33061/testdb
spring.datasource.username=summarydb
spring.datasource.password=Docker#Root99
```

### 3. Run the Application
```bash
# Build and run with Maven
mvn clean install
mvn spring-boot:run
```

The application will start at: **http://localhost:8099**

## ğŸ“¡ API Endpoints

| Method | URL            | Description       | Example Body                                      |
|--------|----------------|-------------------|---------------------------------------------------|
| GET    | /users         | Get all users     | -                                                 |
| GET    | /users/{id}    | Get user by ID    | -                                                 |
| POST   | /users         | Create new user   | `{"username":"john","email":"john@example.com"}` |
| PUT    | /users/{id}    | Update user       | `{"username":"john","email":"john@example.com"}` |
| DELETE | /users/{id}    | Delete user       | -                                                 |

### Example Usage

**Create a user:**
```bash
curl -X POST http://localhost:8099/users \
  -H "Content-Type: application/json" \
  -d '{"username":"john_doe","email":"john@example.com"}'
```

**Get all users:**
```bash
curl http://localhost:8099/users
```

**Get user by ID:**
```bash
curl http://localhost:8099/users/1
```

## ğŸ—„ï¸ Database

The application automatically creates a `users` table with the following structure:

```sql
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    email VARCHAR(100)
);
```

Hibernate will auto-create/update the schema on startup (`spring.jpa.hibernate.ddl-auto=update`).
